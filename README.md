# Blue Nile Machine Intelligence Lab Website

A responsive, modern landing page for the Blue Nile Machine Intelligence Lab (BNMIL). The site is built with semantic HTML, modular CSS, and vanilla JavaScript to deliver a fast, accessible experience.

## Features

- Sticky navigation with smooth scrolling and mobile toggle
- Hero, About, Offerings, Team, Events, and Contact sections
- Intersection Observer powered fade-in animations
- Component-based structure (navbar, footer, feature cards)
- Responsive layouts for desktop, tablet, and mobile

## Project Structure

```
Blue-Nile-Machine-Intelligent-Lab-website/
 index.html

    css/

    fonts/

 README.md
```

## Getting Started

1. Open `index.html` in any modern browser.
2. All assets load locally; no build step is required.
3. To customize content, edit the HTML snippets in `components/` and `sections/`.

## Customization Tips

- Update colors and typography inside `assets/css/style.css`.
- Adjust breakpoints and layout tweaks in `assets/css/responsive.css`.
- Add or remove animations by editing `assets/js/animations.js` and the `data-animate` attributes in section files.
